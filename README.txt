# Copiloto Ruta Escolar

Carpeta lista para subir a cualquier hosting estatico.

## Archivos
- index.html
- manifest.webmanifest
- service-worker.js
- icon-192.png
- icon-512.png

## Como subir
1. Sube toda la carpeta al hosting.
2. Abre `index.html`.
3. En celular, permite GPS.
4. La app guarda todo localmente en el navegador.
5. Usa `Exportar backup` para guardar una copia de seguridad.

## Importante
- No necesita backend.
- WhatsApp se abre con enlaces `wa.me`.
- En Android puede instalarse como app PWA.
- Los datos se guardan en LocalStorage del dispositivo.
